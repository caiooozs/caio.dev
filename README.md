## 📌 Sobre o Projeto

Aplicação web moderna projetada para centralizar a apresentação de competências, principais stacks utilizadas e projetos desenvolvidos ou em andamento. O projeto conta com design responsivo, animações sutis e interface minimalista focada na experiência do usuário.

---

## 🛠️ Stacks e Tecnologias

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
- **UI & Estilização**: [Tailwind CSS v4](https://tailwindcss.com/), [Shadcn UI](https://ui.shadcn.com/), [React Icons](https://react-icons.github.io/react-icons/)
- **Animações**: [Motion](https://motion.dev/)

## ⚙️ Como Executar Localmente

### Pré-requisitos

- **Node.js** (versão 18.x ou superior)
- **npm**, **pnpm** ou **yarn**

### Passo a passo

1. **Clonar o repositório:**
   ```bash
   git clone https://github.com/caiooozs/caio.dev.git
   cd caio.dev
   ```

2. **Instalar as dependências:**
   ```bash
   npm install
   ```

3. **Iniciar o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acessar no navegador:**
   Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o projeto em execução.
